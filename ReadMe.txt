Assignment 5 --- CMPT213

This Program is Designed to Read,Edit and Save command file for Assignment 4.


************Pre-Requrements**********
This program requires JDK1.07 or above for Transparent Swing effects.

This program requires a "offices.txt" file to read&display the offices for selection properly when adding a new Letter or new Package command.

************Other Souces*************
This program uses open-source "beautyeye-3.7" look&feel.

The lib file can be found under ./libs folder and this lib file need to be added to the build path to work

************Features*****************
1. Program will read&load the current command file naming "commands.txt" under the project root folder if available

2. Program supports all commands adding features,such as "New Letter","Deflation" and etc.

3.Program supports editting commands by moving up,moving down,deleting and saving. 

Unfortunatelly no actual editing implemented yet.
